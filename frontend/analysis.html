📘 CDR File Format Guide

This tool analyzes Call Detail Record (CDR) files from different telecom operators and states.

It does not depend on fixed column positions. Instead, it matches your column names with known naming patterns.

⚠️ If a column name does not match any supported pattern, it will be shown as N/A.

✅ Supported File Type

  • CSV file only (.csv)
  • First row must contain column headers
  • UTF-8 encoding recommended

🧠 How Column Detection Works

  • Column order does not matter
  • Column name meaning matters
  • Matching is done using predefined aliases
